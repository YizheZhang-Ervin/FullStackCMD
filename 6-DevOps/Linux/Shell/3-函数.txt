# shell函数
1.系统函数
提取路径名/文件名： basename 字符串/路径名 截取时要去掉的后缀名
根据绝对路径去除文件名得到前缀目录路径： dirname 绝对路径
2.自定义函数
function funName()
{
    命令
    return 返回值
}
调用：funName 参数1 参数2
定义时function和return可有可无
有参参数$#参数个数/$*单字符串显示所有向脚本传递的参数/$$当前进程ID号/$!最后一个进程ID号/$@所有参数但使用时加引号/$?最后命令的退出状态
区别: 外部脚本文件在子shell中运行(独立进程)，shell函数在当前shell进程中运行
